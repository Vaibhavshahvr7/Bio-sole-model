STAGE 6: TEMPORAL PARAMETER CALCULATION
═══════════════════════════════════════════════════════════════════════

  Events extracted from each cycle:

  Cycle Timeline:
  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  R_HS_S      L_FF          L_HS      R_FF                R_HS_E
  Start       Left off      Left      Right               End
  Right       (left foot    strike    off                 (next cycle
  strike      leaves)       (left     (right              starts)
              ← Stance →    foot      foot 
                            lands)    leaves)
              ← Swing →                ← Swing →
  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Calculate Intervals:

  1. Stride Time = R_HS_E - R_HS_S = 1.103 seconds
                                      ↑ One full gait cycle

  2. Stance Time = R_FF - R_HS_S = 0.675 seconds
                                   ↑ Right foot in contact

  3. Swing Time = R_HS_E - R_FF = 0.428 seconds
                                   ↑ Right foot in air

  4. Step Time = L_HS - R_HS_S = 0.551 seconds
                                  ↑ Time to opposite foot contact

  5. Cadence = 60 / Step_Time = 108.9 steps/minute

  6. Double Support = (L_FF - R_HS_S) + (R_FF - L_HS)
                    = 0.146 + 0.151 = 0.297 seconds

  7. Single Support = L_FF - L_HS = 0.378 seconds
                                    ↑ Left foot in air

  8. Stance % = (Stance Time / Stride Time) × 100 = 61.2%

  9. Swing % = (Swing Time / Stride Time) × 100 = 38.8%
