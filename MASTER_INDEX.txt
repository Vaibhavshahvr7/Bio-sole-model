KEY CONCEPTS AT A GLANCE
════════════════════════════════════════════════════════════════════════

PIPELINE STAGES:
  1. Load Data              → Organize sensor data from files
  2. Window & Normalize     → Convert to fixed-size windows
  3. Gait Classification    → Filter real gait from non-gait
  4. Heel Strike Detection  → Locate foot contact events
  5. Phase Extraction       → Normalize cycle length
  6. Parameter Calculation  → Compute clinical metrics

MAIN OUTPUT (data_final):
  • Cadence: Steps per minute
  • Stride Time: Full gait cycle duration
  • Stance Time: Foot in contact
  • Swing Time: Foot in air
  • Stance %: Percentage of cycle in stance
  • Swing %: Percentage of cycle in swing

CLINICAL SIGNIFICANCE:
  ✓ Healthy: Cadence 100-130, Stance 60%, Swing 40%
  ✗ Abnormal: Low cadence, high variability, asymmetry

SENSOR DATA SPECS
════════════════════════════════════════════════════════════════════════

14 Sensor Channels:

LEFT FOOT (7):
  [0-2] Acceleration (X, Y, Z) in g
  [3-5] Angular velocity (X, Y, Z) in dps
  [6]   Vertical force in N

RIGHT FOOT (7):
  [7-9]   Acceleration (X, Y, Z) in g
  [10-12] Angular velocity (X, Y, Z) in dps
  [13]    Vertical force in N

Sample Rate: 100 Hz (10 ms per sample)
Recording Duration: Variable (usually 1-5 minutes)
Typical Sample Count: 10,000-30,000 per subject
